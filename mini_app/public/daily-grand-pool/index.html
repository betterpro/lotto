Daily Grand

Daily Grand pool guide

Daily Grand pools follow the same trust principles as any lottery pool: define the shares, buy the tickets, show the proof, and split any prize according to the record that existed before the draw.

When Daily Grand fits a pool

Daily Grand can work well for groups that like recurring national draws and want a simple share-based structure. The game is different from Lotto Max and Lotto 6/49, so it should have its own round records.

Write the round rules first

Members should know the share cost, the draw date, the total number of shares, and how any prize type will be handled. Written rules are especially important when a group plays several different lottery games.

Ticket proof and result checking

Post ticket photos before the draw. After the draw, compare the tickets with the published result and record the outcome for the group, even when there is no prize.

Repeat rounds

If a group plays Daily Grand regularly, auto-participation can reduce admin work. Members should still be able to see which round they entered and how many shares they own.

LottoChee separates each Daily Grand round so the group can keep participation and ticket proof organized.

A Daily Grand pool can lose. Set a budget and treat each draw as entertainment.
